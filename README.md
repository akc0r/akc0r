## About

I'm a **Software Engineer** building production-grade applications end to end — from reactive front-ends (Next.js / Tailwind) to scalable backends and services.

- I work across the **full stack**, with systems-level precision (C / C++) and clean, maintainable services.
- I care about **performance, reliability, and developer experience** as much as shipping features.
- On the **AI** side, I integrate machine learning into real products to deliver measurable value.

My focus: building software that is **fast, robust, and a pleasure to use**.
